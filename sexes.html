Alternative sex tagging system for erotic and pornographic artwork

Dear friends of porn and social justice

There have been many discussions about how to avoid potentially misgendering and/or problematic terms like "Futa", "Shemale", "Dickgirl", "Hermaphrodite", "Cuntboy" and even "male" or "female" while trying to describe the specific set of genitals a character has. Here I want to propose an alternative.

The basics

Each component of the possible genitals gets an own abbreviations.
The abbreviations chosen for this example
Component Abbreviations
Penis pe
Testes te
Breasts ma
Vagina va
Clitoris cli
A composed abbreviation of these elements is than used to describe a set of genitals. Components that are not included in a character get a prefix, for example a simple "no" or maybe a "!" or a "~". A prefix for included components may be useful in case a parser is used. The system would allow to avoid commonly used character tropes and also propose a certain degree of additional freedom. Wheter the system is to be used only for visible components or also for internal ones, like testes for reptilian characters is up to diskussion. Generally I would disrecommend.

Possible implementation

I would suggest an implementation via an implication table. Implication table work pretty much like this: The user searches for a term, for example "vaginal". The database looks up the implikation table and puts out all submissions tagged with "vaginal" but also every submission tagged with a tag that implies "vaginal", like for example submission tagged with "vaginal_penetration" or "vaginal_creampie".
